Frequently Asked Questions

How does Unit 1 free access work?
Unit 1 (Atomic Structure & Properties) is completely free — no account, no credit card, no signup required. You can access all Unit 1 lessons and practice questions right away at apchem.mindspirelab.com.
What is the custom test builder?
Like SaveMyExams, you can select which units/topics to include, choose MCQ or FRQ or both, set the number of questions, and generate a timed custom practice test. It's available on paid plans.
Are these real College Board questions?
The platform includes questions modeled directly on past AP Chemistry exams (2021–2025), written in the exact College Board style. Some actual PYQ are included as well as new questions following the same format.
How do I get my access code after paying?
After payment via WhatsApp, Faridah will send you a personal access code within 24 hours. Use this code to unlock paid content on the platform.
